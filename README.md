# 💼 AI Job Prediction System

## 📌 Project Overview
This project is an AI-based web application that predicts suitable job roles based on user inputs like:
- Degree
- Specialization
- CGPA

It helps students choose the right career path using machine learning.

---

## ⚙️ Technologies Used
- Python 🐍
- Machine Learning (Scikit-learn)
- Pandas & NumPy
- Streamlit (for UI)

---

## 📊 Features
- Predicts job roles based on user input
- Easy-to-use interface
- Fast and accurate predictions
- Real-world dataset used

---

## 📁 Project Structure
- `app.py` → Main application
- `model_train.py` → Model training code
- `job_dataset.csv` → Dataset
- `.pkl files` → Trained model & encoders
